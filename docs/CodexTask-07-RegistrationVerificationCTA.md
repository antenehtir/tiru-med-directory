# Codex Task 07: Registration and Verification CTA System

## Project

DigitalDirectory-v2

## Goal

Improve the provider registration and verification call-to-action system for DigitalDirectory-v2.

This task should make the homepage communicate a clear and trustworthy path for doctors, facilities, pharmacies, diagnostic centers, and other healthcare providers to request listing or verification.

This task is frontend-only.

Do not add backend functionality.

## Main Objective

Create a polished registration and verification CTA system that helps providers understand:

1. Who can register
2. What they can request
3. Why verification matters
4. What the verification process looks like
5. What action to take next

The system must feel:

- Trustworthy
- Professional
- Clear
- Healthcare-specific
- Mobile-friendly
- Not overwhelming
