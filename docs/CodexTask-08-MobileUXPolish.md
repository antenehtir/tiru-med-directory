## Main Objective

Improve the mobile experience without changing product scope.

Focus on:

1. Mobile spacing
2. Mobile typography
3. Touch target sizes
4. Search box responsiveness
5. Card stacking
6. Doctor card name wrapping
7. Facility card readability
8. Registration CTA mobile layout
9. Bottom navigation comfort
10. Scroll comfort

---

## Specific Issues to Review and Improve

### 1. Hero Section Mobile Layout

On small screens:

- Headline should not be too large.
- Text should wrap naturally.
- Search box should not feel cramped.
- CTA buttons should stack cleanly if needed.
- Search chips should wrap without horizontal overflow.

---

### 2. Search UI Mobile Polish

Improve:

- Search input height
- Search button placement
- Filter chip wrapping
- Popular search chip spacing
- Location row stacking
- Empty state preview spacing

Search should remain the most important visual element.

---

### 3. Facility Cards Mobile Polish

Facility cards should:

- Stack cleanly on mobile
- Keep verification badge visible
- Keep action buttons tappable
- Avoid text overflow
- Use consistent spacing
- Keep services readable

Action buttons should be at least 44px high.

---
